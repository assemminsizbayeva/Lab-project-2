Exercise 1.1: Basic Syntax

Section I: Programming Fundamentals

Problem Statement

Write a program that demonstrates basic programming syntax and structure.

Solution

// Basic syntax example
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
        int number = 42;
        System.out.println("The answer is: " + number);
    }
}

Explanation

This solution demonstrates fundamental programming concepts including class declaration, main method structure, variable declaration, and basic output operations.
